Customer Segmentation ML

Machine Learning model for customer behavior analysis using K-means clustering

Python Scikit-learn Pandas Matplotlib Seaborn

Project Overview

Developed a comprehensive customer segmentation system using unsupervised machine learning to identify distinct customer groups based on purchasing behavior and demographics.

Key Features

  • K-means clustering algorithm implementation
  • Data preprocessing and feature engineering
  • Optimal cluster determination using elbow method
  • Interactive visualization of customer segments
  • Customer profiling and behavior analysis

Technical Implementation

import pandas as pd
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt
import seaborn as sns

# Data preprocessing
data = pd.read_csv('customer_data.csv')
features = ['annual_income', 'spending_score', 'age']
X = data[features]

# Feature scaling
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# K-means clustering
kmeans = KMeans(n_clusters=5, random_state=42)
clusters = kmeans.fit_predict(X_scaled)

# Visualization
plt.figure(figsize=(10, 6))
sns.scatterplot(x=X[:, 0], y=X[:, 1], hue=clusters, palette='viridis')
plt.title('Customer Segments')
plt.show()
